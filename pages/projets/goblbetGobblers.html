Première version

Cette première version est une application console développée en langage C lors d'une S.A.E. en première année de but informatique par équipe de 2.

Seconde version

J'ai par la suite décidé de recommencer ce projet en le faisant en JavaScript en utilisant du HTML et CSS pour l'interface utilisateur.

Cela a permis de m'améliorer en JavaScript

  • Mise en œuvre d'un MVC : Organisation plus claire du code et séparation des responsabilités ;
